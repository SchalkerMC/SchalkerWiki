При помощи этого плагина вы сможете ощутить себя на уроке физкультуры. Мячик можно кидать, отбивать, подавать  в общем всё, что нужно. В плагине присутствует физика и гравитация.

## Использование

| Кнопка | Назначение |
| --- | --- |
| <kbd>Q</kbd> | Кинуть мяч с минимальной силой |
| <kbd>БЕГ + ЛКМ</kbd> | Кинуть мяч сильнее |
| <kbd>SHIFT + ЛКМ</kbd> | Подкинуть мяч на месте |
| <kbd>ПКМ + ПУСТАЯ РУКА</kbd> | Подобрать мяч |
| <kbd>ПКМ</kbd> | Поставить мяч |

## Команды

| Команда | Назначение |
| --- | --- |
| `/ball skin <КОД ГОЛОВЫ>` | Поменять скин мячика. Код головы  Mojang (base64) |
