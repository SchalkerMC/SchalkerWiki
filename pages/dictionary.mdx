# Словарь терминов

Основные термины и понятия, используемые на сервере.

## Термины

- **Вайтлист** - список игроков, которым разрешен вход на сервер
- **Гильдия** - объединение игроков с общими целями
- **Спавн** - точка появления на сервере
- **Приват** - защита территории от других игроков

_Список терминов будет дополнен_
