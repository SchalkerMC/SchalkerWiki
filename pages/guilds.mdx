### Создание клана (через чат)

1. Откройте меню командой <code>/clan</code> и нажмите «Создать клан», либо используйте <code>/clan create</code>.
2. Введите тег в чат. Тег должен быть уникальным и без пробелов.
3. Введите название клана в чат (можно цвета, если разрешено).
4. После сообщения об успехе клан создан.
5. Для отмены в любой момент напишите <code>cancel</code> или <code>отмена</code>.

![Создание клана](clans/clans-create.png)

### Приватность клана

- **public** — любой игрок может вступить через профиль клана.
- **invite_only** — вступление только по приглашению.
- **private** — клан скрыт в списке, профиль виден только участникам.

Переключение доступно в настройках GUI или командой <code>/clan privacy</code>.

### Приглашения

- Приглашение отправляет игрок с правом <em>Invite Members</em>.
- Игрок получает кликабельное сообщение с кнопками «Принять» и «Отклонить».
- Приглашение автоматически истекает по таймеру из конфига.

### Профиль клана

Профиль показывает:
- тег и название;
- дату создания;
- приватность;
- описание;
- список участников (частично);
- главу клана;
- кнопку «Вступить» для публичных кланов.

### Настройки клана (GUI)

В разделе настроек доступны:
- смена приватности;
- смена названия и тега (через ввод в чат);
- описание клана;
- цвет баннера;
- включение/выключение показа тега;
- публичность профиля;
- Friendly Fire (огонь по своим).

![Настройки клана](clans/clans-settings.png)

### Клановый чат

- Включается/выключается лидером командой <code>/clan chat</code>.
- Сообщения отправляются с префиксом из конфига (по умолчанию <code>*</code>).
- Формат сообщений настраивается в messages.yml.

Пример: <code>* привет клану</code>

### Роли и права

Роли по умолчанию:
- **Leader** — все права.
- **Co-Leader** — все права, кроме расформирования.
- **Moderator** — приглашение и кик.
- **Member** — базовый участник.

Кастомные роли:
- создаются командой <code>/clan role create</code>;
- приоритет 1–100 (чем выше — тем больше прав);
- удаление роли переводит участников в базовую роль Member.

В GUI «Управление ролями» можно назначать роли участникам.

### Friendly Fire

Если Friendly Fire выключен, урон между участниками одного клана блокируется (включая урон от стрел).

### Быстрые ответы на типовые вопросы

- **Почему не могу вступить?**
  Клан приватный, переполнен или у вас уже есть клан.
- **Почему не могу кикнуть?**
  Нужны права, и вы не можете кикать равных/выше по роли.
- **Почему не вижу клан в списке?**
  Клан приватный или профиль закрыт для всех.
