# Команды

Здесь будет список всех доступных команд на сервере.

## Основные команды

| Команда | Описание |
|---------|----------|
| `/spawn` | Телепортация на спавн |
| `/clan` | Меню гильдий |
| `/register` | Регистрация аккаунта |
| `/login` | Вход в аккаунт |

_Список команд будет дополнен_
